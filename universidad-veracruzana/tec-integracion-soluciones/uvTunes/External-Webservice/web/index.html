uvTunes Web Services

Este es el sitio para acceder a los servicios de la uvTunes.

Operación: Consultar estoque


Esa operación regresa todos los articulos y sus cantidades en el stock de uvTunes.
El proveedor envía una petición GET para la URL https://uvtunes-external-webservice.azurewebsites.net/external_webservice/servicios/proveedores con su credencial, y el sistema regresa un JSON con el exito de la operación y un arreglo con los productos.
Las variables deben ser enviadas como parámetros en la URL.

El proveedor envía a la uvTunes:

  • string: Credencial del proveedor


El proveedor recibe de la uvTunes:

  • boolean: Éxito de la operación
  • array: Productos en estoque
    • int: Id del producto (id común entre las bases)
    • string: Nombre del producto
    • int: Cantidad solicitada del producto


Ejemplo de comunicación

1. El proveedor envia: proveedores?credencial=1MO2FPFPOMFMP
2. El web service retorna: { “exito”: true,
  “productos”: [
      { “id”: 5, “nombre”: “Shatter Me”, “cantidad”: 2 },
      { “id”: 21, “nombre”: “Uncharted”, “cantidad”: 1 },
      { “id”: 26, “nombre”: “Thor”, “cantidad”: 3 }
  ]}

Operación: Hacer sugerencia


Esa operación recibe y guarda una sugerencia hecha por el proveedor.
El proveedor envía una petición POST para la URL https://uvtunes-external-webservice.azurewebsites.net/external_webservice/servicios/proveedores con su credencial y la sugerencia, y el sistema regresa un JSON con el exito de la operación.
Las variables deben ser enviadas como parámetros en la URL.

El proveedor envía a la uvTunes:

  • string: Credencial del proveedor
  • string: Sugerencia que desea enviar


El proveedor recibe de la uvTunes:

  • boolean: Éxito de la operación


Ejemplo de comunicación

1. El proveedor envia: proveedores?credencial=1MO2FPFPOMFMP&sugerencia=<Sugerencia>
2. El web service retorna: { “exito”: true }
